Project Analysis

Advanced Data Structure Visualizer

Qt Widgets • C++17 • CMake • Animated visualizations for learning data structures

Overview

A Qt Widgets application organized around a QStackedWidget flow: landing → menu → theory → operations → visualization. The implemented visualization covers Binary Tree insertion with animated traversal and drawing using QPainter.

Target
AdvDS (CMake)
Languages
C++17
Framework
Qt 6/5 Widgets
Build
CMake with AUTOUIC/AUTOMOC/AUTORCC

Key Entry Points

  • main: creates QStackedWidget, wires navigation between pages.
  • Pages: HomePage, MenuPage, TheoryPage, OperationPage, TreeInsertion.
  • UI approach: Pure code-driven widgets and custom painting; mainwindow.ui exists but not used.

Navigation Flow (signals/slots)

  • HomePage::navigateToMenu → Menu
  • MenuPage::dataStructureSelected(ds) → creates TheoryPage(ds)
  • TheoryPage::tryItYourself → creates OperationPage(ds)
  • OperationPage::operationSelected("Insertion") with ds == Binary Tree → creates TreeInsertion
  • Each page exposes a back signal to return to the previous view.

Implemented Visualization

Binary Tree — Insertion

  • Interactive controls: input value, Insert, Clear Tree.
  • Animated traversal path highlighting with timed steps.
  • Dynamic layout: node positions computed per level; edges and nodes drawn with QPainter.
  • Status text updates (e.g., comparisons, insert locations).

Files of Interest

  • CMakeLists.txt: Qt Widgets target, adds all page sources.
  • main.cpp: Stacked navigation and signal wiring.
  • homepage.*, menupage.*: Landing and menu UIs with subtle animations.
  • theorypage.*: Theory content generator per data structure.
  • operationpage.*: Operation selections and transitions.
  • treeinsertion.*: Visualization implementation for insertion.

Notable Observations

  • Only Binary Tree → Insertion visualization is implemented; others are TODO.
  • MainWindow and mainwindow.ui are present but unused by main.cpp.
  • homepagewidget.cpp references a missing header and is not included in the build.

How to Build/Run

  • Open in Qt Creator (Kit: Qt 6/5 Widgets) and build target AdvDS.
  • Or generate with CMake and build with your compiler toolchain.
